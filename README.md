# Urinalysis Case Study: Alkaptonuria

## Case Overview

A 45-year-old patient presents with darkening urine that turns black upon standing. Your task is to analyze the provided data to determine the cause.

## Repository Structure

- `patient_data.csv`: Simulated quantitative urinalysis results.
- `submission_template.md`: Worksheet for answering case study questions.
- `PULL_REQUEST_TEMPLATE.md`: Template for submitting your work.

## Instructions for Students

1. Fork this repository.
2. Create a new branch named `student-[your-name]-submission`.
3. Fill out the `submission_template.md` file with your answers.
4. Open a Pull Request to submit your work for review.
